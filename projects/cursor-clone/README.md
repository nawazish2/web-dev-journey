# Cursor Landing Recreation

This project recreates the current Cursor homepage in a desktop-first static format using only HTML and CSS.

## Live demo

- https://cursor-landing-neon.vercel.app/

## Sections recreated

- Header with product/resources navigation and CTAs
- Hero section with large centered headline and product demo
- Trusted-by logo strip
- Product showcase blocks for Agents, Surfaces, and Tab
- Testimonial quote grid
- "Stay on the frontier" feature card section
- Changelog block
- Applied research team / about section
- Recent highlights cards
- Final CTA
- Footer with product, resources, company, legal, and connect links

## Fonts and colors

- Font: `Inter`
- Main background: `#09090b`
- Panels: `#0f1014`, `#15161b`, `#17181d`
- Primary text: `#f5f5f6`
- Secondary text: `#a1a1aa`
- Accent button: `#ffffff`

## Assets

- Brand-inspired visuals are recreated with HTML and CSS panels.
- Logos, avatars, and some imagery load from public Cursor asset URLs to stay closer to the reference.

## Files

- `index.html`
- `styles.css`

## Notes

- Desktop-only layout, per assignment requirement.
- No JavaScript, no TailwindCSS, and no animations.
